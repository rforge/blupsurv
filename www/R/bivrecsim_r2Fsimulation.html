/simulation [ Modules ]

NAME

    simulation

FUNCTION

Routines to generate simulated bivariate data. None of these are user-facing, they were only used in simulations.

The function bivrecsim is the main simulation routine that conducts a set of simulations and dumps the output to a file. The simulation output must then be parsed separately.

CONTENTS

    gen2AG --- create AG data frame
    generatecensoring --- generate censoring times
    generatecovariate --- generate simulated covariates
    generatefrailty --- simulate frailties
    generaterecurrent --- generate simulated recurrent event times
    initsetup  --- intial setup for simulation
    L --- stratification function
