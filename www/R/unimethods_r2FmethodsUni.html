NAME

    methodsUni --- methods for univariate recurrent event processes

FUNCTION

S3 methods for univariate recurrent events. These work analogously to those in methodsBiv. Again, the method dispatcher unirec will typically call unirec.formula, which constructs a data frame as needed by bivrec.agdata (with a dummy second process).
